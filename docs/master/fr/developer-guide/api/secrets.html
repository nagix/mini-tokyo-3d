# Secrets

L'objet Secrets est un objet qui stocke les jetons d'accès utilisés pour récupérer les données et est défini sur l'option du constructeur Map secrets.

Type Object (opens new window)

# Propriétés

# challenge2024 (string (opens new window))

Jeton d'accès pour le Open Data Challenge for Public Transportation 2024 (opens new window). S’il n’est pas spécifié, le jeton par défaut sera utilisé.

Avertissement

En raison de la fin de l'Open Data Challenge for Public Transportation 2024, ce paramètre n'est plus utilisé à partir de la version 4.0.

# challenge2025 (string (opens new window))

Jeton d'accès pour le Open Data Challenge for Public Transportation 2025 (opens new window). S’il n’est pas spécifié, le jeton par défaut sera utilisé.

Avertissement

En raison de la fin de l'Open Data Challenge for Public Transportation 2025, ce paramètre n'est plus utilisé à partir de la version 4.0.

# odpt (string (opens new window))

Jeton d'accès pour le Public Transportation Open Data Center (opens new window). S’il n’est pas spécifié, le jeton par défaut sera utilisé.

# tokyochallenge (string (opens new window))

Jeton d'accès pour le Open Data Challenge for Public Transportation in Tokyo (opens new window). S’il n’est pas spécifié, le jeton par défaut sera utilisé.

Avertissement

En raison de la fin de l'Open Data Challenge for Public Transportation à Tokyo, ce paramètre n'est plus utilisé à partir de la version 3.1.
